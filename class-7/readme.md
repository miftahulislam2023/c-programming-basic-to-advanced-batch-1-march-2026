# Class - 7

## Meanings of Size of Variables
- char - 1byte - 8bit -> ২৫৬টি ভিন্ন ভিন্ন জিনিস ধারণ করতে পারে। কিন্তু একবারে কেবল একটি। সেটা হলো বর্ণ, letter, digit etc
  - 1bit -> 2 -> 2^1
  - 2bit -> 4 -> 2^2
  - 3bit -> 8 -> 2^3
  - 4bit -> 16 -> 2^4
  - nbit -> 2^n -> 2^n (2 power n)
  - 8bit -> 2^8 -> 256
  - 0 -> 48
  - a -> 97
  - A -> 65
- int -> 4byte -> 2^32 -> 4,29,49,67,296
- float -> 4byte -> 2^32
- double -> 8byte -> 2^64 -> 18,446,744,073,709,551,616

## Format specifier
- %d -> int
- %c -> char
- %f -> float
- %lf -> double
