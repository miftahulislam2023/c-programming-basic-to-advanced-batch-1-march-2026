## Naming Convention
1. নাম শুধুমাত্র ইংরেজি upper এবং lowercase letter, 0-9 পর্যন্ত digit এবং underscore দিয়ে তৈরি হতে হবে
2. কখনো digit দিয়ে শুরু হতে পারবে না
3. C হলো case-sensitive, যার মানে C upper এবং lowercase letter-এর মধ্যে পার্থক্য করতে পারে: তাই C-এর ক্ষেত্রে Myfunc এবং myfunc সম্পূর্ণ আলাদা দুটি নাম।
